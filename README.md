# Детекция футбольного мяча с использованием YOLOv8

## Описание
Проект позволяет обнаруживать футбольный мяч на изображениях с помощью предобученной нейронной сети YOLOv8.  
Функционал:

- Детекция мяча с координатами bounding box и confidence
- Веб-интерфейс для загрузки изображений и визуализации результатов
- История обработанных изображений
- Отображение статистики и генерация отчётов PDF/Excel
- Удобный интерфейс с понятными результатами и пояснениями

---

## Требования
- Python 3.10+  
- Установленные пакеты (через `requirements.txt`):
  - flask
  - ultralytics
  - opencv-python
  - pandas
  - fpdf
  - openpyxl

---

## Как запустить

1. Клонировать репозиторий:

```bash
git clone https://github.com/dalios009/football_ball_detection.git
```
2. Перейти в папку проекта:
```bash
cd football_ball_detection
```
3. Создать и активировать виртуальное окружение:
```bash
python -m venv venv
venv\Scripts\activate   # Windows
```
4. Установить зависимости:
 ```bash
   pip install -r requirements.txt
 ```
6. Запустить Flask веб-приложение:
 ```bash
   python app.py
```
8. Открыть браузер и перейти по адресу:
```bash
  http://127.0.0.1:5000
 ```
10. Загрузить изображение футбольного матча и нажать "Запустить анализ", чтобы увидеть результаты детекции.
